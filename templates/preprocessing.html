Preprocessing

Preprocessing dalam pengolahan data adalah serangkaian langkah atau teknik yang digunakan untuk membersihkan, mengubah, dan mempersiapkan data mentah menjadi bentuk yang lebih sesuai untuk analisis atau pemodelan data. Tujuan utama dari preprocessing adalah untuk meningkatkan kualitas data, menghilangkan noise atau gangguan, serta membuat data siap untuk diproses oleh algoritma pembelajaran mesin atau model statistik.

Mengidentifikasi dan Mengatasi Missing Value:

Setelah melakukan analisis data, kami berhasil mengidentifikasi keberadaan missing values dalam dataset. Ditemukan bahwa beberapa atribut memiliki missing values, seperti "tidak dilakukan" untuk atribut foto toraks, "tidak diketahui" untuk atribut riwayat diabetes, "tidak diketahui" untuk atribut status HIV, dan "tidak dilakukan" untuk atribut hasil TCM.

Untuk mengatasi masalah ini, kami menggunakan metode imputasi dengan menerapkan modus, yaitu nilai yang paling sering muncul dalam setiap atribut. Dengan pendekatan ini, kami memastikan bahwa dataset menjadi lebih lengkap dan akurat untuk analisis lebih lanjut.

Model machine learning tidak dapat memproses data berbentuk teks secara langsung. Pada dataset TB yang kita miliki, terdapat beberapa data dalam bentuk teks. Oleh karena itu, perlu dilakukan transformasi pada dataset agar model dapat memprosesnya dengan benar. Kami melakukan transformasi ini pada fitur jenis kelamin, foto toraks, status HIV, riwayat diabetes, lokasi anatomi, serta kolom kelas.
